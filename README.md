# Brand-nyamhap-db-project2
Repositori ini merupakan lanjutan dari proyek basis data terintegrasi untuk Brand NyamHap, sebuah bisnis FnB yang berfokus pada efisiensi operasional melalui sistem digital. Proyek ini dikembangkan oleh mahasiswa IPB dari program studi Statistika dan Data Sains sebagai bagian dari pembelajaran sistem basis data dan implementasi CRUD berbasis web.
